# Migzi.ai — WebGPU‑Powered 3D Document Platform

Migzi.ai is a next‑generation browser‑based 3D platform that allows users to import **PDFs and PowerPoints** into immersive environments.  
It is built for high‑performance rendering using **WebGPU**, **Three.js**, and a custom rendering pipeline designed for modern GPUs.

---

## 🚀 Features

- **WebGPU Rendering Pipeline**  
  High‑performance rendering with compute shaders, GPU buffers, and custom pipelines.

- **3D Document Viewer**  
  Import PDFs and PowerPoints directly into a 3D scene.

- **React + Three.js Frontend**  
  Modular, component‑driven architecture for UI and scene management.

- **Node.js Backend**  
  Handles file processing, conversion, and asset delivery.

- **Optimised Scene Graph**  
  Designed for large documents and multi‑page environments.

---

## 🧱 Architecture Overview

Migzi.ai is split into two main layers:

### Frontend (React + Three.js + WebGPU)
- Scene management  
- GPU pipelines  
- UI/UX  
- Document rendering  
- Interaction systems  

### Backend (Node.js + SQL)
- File upload & processing  
- PDF/PPT conversion  
- Asset caching  
- User/session management
